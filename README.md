# 🌸 Iris Flower Classification using KNN and Streamlit

A Machine Learning web application that predicts the species of an Iris flower using the K-Nearest Neighbors (KNN) algorithm. This project is built with Python and deployed using Streamlit, providing an interactive and user-friendly interface for real-time predictions.

---

## 🚀 Live Demo

https://sanjula2003-iris-ml-app-app-zvf0ml.streamlit.app

---

## 📌 Project Overview

This project uses a trained KNN Machine Learning model to classify Iris flowers into three species based on four input features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Users can adjust sliders in the UI and instantly receive predictions with an attractive interface.

---

## 🌼 Iris Flower Species

The model predicts one of the following flower species:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

## ⚙️ Installation

Clone the repository:

git clone https://github.com/your-username/iris-ml-app.git

```
Move into the project folder:

cd iris-ml-app

```
Install required packages:

pip install -r requirements.txt

```
Run the Streamlit app:

streamlit run app.py

```
